# Text Elements
MODULAR ARITHMETIC ^aDYhZnER

Modular arithmetic is a system of arithmetic for integers, where numbers 
"wrap around" when reaching a certain value, called the mod(ulus).

Under modular arithmetic (with mod N), the only numbers are 0, 1, 2, 3, ..., N - 1,
and they are known as residues modulo N.

DEFINITIONS

a   b (mod n) if and only if n  a - b (n divides a - b)
a, b, and n are integers; n > 1 ^m3TgniTf
